## 🛑 경고 (Disclaimer)
* **개발 능력치:** $0.001$ (존재하지 않는 수준)
* **코드 상태:** 내 컴퓨터에선 분명 돌아갔음. 솔직히 왜 되는지 모르겠음 (기적에 의존 중).
* **뇌 메모리 상태:** 알고리즘 문제만 보면 `NullPointerException` 발생 후 셧다운.
