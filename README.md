# 📡 SynoMonitor (for macOS)

**SynoMonitor**는 Mac의 상단 메뉴바에 상주하며 Synology NAS의 상태를 실시간으로 모니터링할 수 있는 빠르고 가벼운 macOS 전용 유틸리티 앱입니다. 

번거롭게 웹 브라우저를 열고 DSM에 로그인할 필요 없이, 단축키 한 번으로 현재 NAS의 네트워크 속도와 상태를 즉시 확인할 수 있습니다.

## ✨ 주요 기능 (Features)

- **메뉴바 실시간 위젯**: 상단 메뉴바에 찰싹 붙어서 NAS의 현재 다운로드/업로드 속도를 실시간으로 표시합니다. (Mac의 네트워크 속도로 변경하여 표시할 수도 있습니다.)
- **글로벌 단축키 지원**: 언제 어디서든 `Cmd + Option + N` 키를 누르면 팝오버 창이 나타나 상세 정보를 보여줍니다.
- **스마트 폴링 (Smart Polling)**: 창이 열려 있을 때는 3초 간격으로 빠르게 데이터를 가져오고, 창이 닫히면 15초 간격으로 전환하여 Mac의 배터리와 리소스를 아낍니다.
- **절전 모드 완벽 대응**: Mac이 잠자기에 들어가면 모니터링을 완전히 멈추고, 깨어나면 자동으로 다시 시작합니다.
- **Tailscale 완벽 지원**: 외부망에서도 VPN(Tailscale 등)을 통해 안전하게 NAS 상태를 체크할 수 있습니다. 연결 상태에 따라 메뉴바 아이콘이 직관적으로 변경됩니다.

## 💻 설치 및 실행 방법 (Getting Started)

1. GitHub의 **Releases** 탭에서 최신 버전의 `SynoMonitor.app.zip` (또는 `.dmg`) 파일을 다운로드합니다.
2. 압축을 해제하고 `SynoMonitor.app`을 **응용 프로그램 (Applications)** 폴더로 이동합니다.
3. 앱을 실행하면 상단 메뉴바에 네트워크 아이콘이 나타납니다.
4. 메뉴바 아이콘을 클릭한 뒤, 우측 상단의 톱니바퀴 아이콘을 눌러 **NAS IP주소, 포트, 계정 정보**를 입력하면 즉시 연결됩니다!

## ⚙️ 개발 환경 (Development)

이 프로젝트는 100% Swift와 SwiftUI로 작성되었습니다.

- **macOS Target**: macOS 13.0 이상
- **프레임워크**: SwiftUI, Combine, Carbon (글로벌 단축키 감지)
